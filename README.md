# Personality Classification Project 

An end-to-end Machine Learning pipeline developed to classify individual personality types from behavioral or textual data. This project leverages data preprocessing, feature engineering, and supervised machine learning algorithms to uncover patterns and predict personality traits accurately.

---

##  Project Overview
Understanding human personality through data has extensive applications in HR recruitment, targeted marketing, psychology, and personalized user experiences. This project aims to build a robust classification model capable of analyzing the provided dataset and sorting individuals into distinct personality groups.

##  Repository Structure
*   **`ML PROJECT PERSONALITY CLASSIFICATION  (1).ipynb`**: The primary Jupyter Notebook containing the entire machine learning workflow (EDA, preprocessing, training, and evaluation).
*   **`personality_dataset (3).xls`**: The source dataset used for training and evaluating the classification models.

##  Tech Stack & Dependencies
This project is built using Python and standard data science libraries. To run the notebook, you will need:
*   Python 3.x
*   NumPy
*   Pandas
*   Scikit-Learn
*   Matplotlib / Seaborn (for data visualization)
*   xlrd / openpyxl (for reading `.xls` files)

##  Workflow
1.  **Data Ingestion & EDA**: Loading the Excel dataset and analyzing distributions, missing values, and feature correlations.
2.  **Data Preprocessing**: Handling missing text or numerical data, cleaning text (if applicable), and scaling features.
3.  **Feature Engineering**: Converting raw attributes into meaningful numerical representations optimized for machine learning algorithms.
4.  **Model Selection & Training**: Training multiple classification algorithms (e.g., Logistic Regression, Random Forests, SVM, or Gradient Boosting) to compare performance.
5.  **Evaluation**: Assessing models using metrics like Accuracy, Precision, Recall, and F1-Score.
